# E-Commerce-Customer-Segmentation-and-Purchase-Prediction
Machine Learning project for E-Commerce Customer Segmentation and Purchase Prediction using RFM Analysis, Clustering (K-Means, Hierarchical, DBSCAN), Random Forest Classification, and Power BI Dashboard.

# 🛒 E-Commerce Customer Segmentation & Purchase Prediction

## Overview

This project uses Machine Learning and RFM (Recency, Frequency, Monetary) Analysis to segment e-commerce customers based on purchasing behavior and predict future customer actions. The goal is to help businesses improve customer retention, personalize marketing campaigns, and increase revenue.

## Objectives

* Identify high-value, loyal, and at-risk customers
* Segment customers using clustering techniques
* Predict customer behavior using machine learning models
* Generate actionable business insights through Power BI

## Technologies Used

* Python
* Pandas, NumPy
* Scikit-Learn
* Matplotlib, Seaborn
* Power BI
* Jupyter Notebook

## Models Used

### Customer Segmentation

* K-Means Clustering
* Hierarchical Clustering
* DBSCAN

### Prediction Models

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier

## Key Results

* Best Clustering Model: **K-Means** (Silhouette Score: 0.34)
* Best Prediction Model: **Random Forest**
* Accuracy Achieved: **98.27%**
* Cross-Validation Accuracy: **97.8%**

* Customer Segment Analysis

## Business Impact
